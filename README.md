## To run

Requisitos:
- Ter o PHP instalado.

### Passos:

#### No terminal:

``` git clone  https://github.com/LpxsBr/freelas.git ```

``` cd freelas ```

obs: para rodar, foi criado um prefixo "shape"

para printar a cruz no terminal

``` php shape cross ```

para printar x no terminal

``` php shape x ```

#### No browser

use o comando:

``` php shape browser ```

depois abra no browser
